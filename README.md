# TennisSquash (Unity)

Учебный проект по предмету «Программирование игр»: аркадный тренажер ударов по стене в стиле сквоша/тенниса.

## Что реализовано

- Управление игроком с клавиатуры (`WASD` или стрелки).
- Два типа ударов:
  - `F` — топспин (более высокий и стабильный полет),
  - `E` — плоский удар (быстрее и ниже).
- Удар по мячу при нахождении в радиусе удара, а также автоудар при касании триггера игрока.
- Система очков:
  - `+1` за отскок от стены, если до этого мяч не попал в бонусную мишень,
  - `+bonusScore` за попадание в бонусную цель (`TargetCircle`).
- Бонусная мишень:
  - краткая визуальная вспышка при попадании,
  - случайное перемещение по стене после срабатывания.
- Сброс серии и мяча при ошибке:
  - мяч улетел за игрока,
  - мяч упал под пол,
  - мяч остановился слишком надолго.
- Визуальный milestone: при достижении 10 очков меняется материал стены, при сбросе очков материал возвращается.
- Отображение текущего счета в UI (`TextMeshPro`).

## Структура скриптов

- `Assets/Scripts/Player.cs` — движение игрока, выбор типа удара и логика удара по мячу.
- `Assets/Scripts/ShotManager.cs` — параметры типов ударов (топспин/плоский).
- `Assets/Scripts/Ball.cs` — поведение мяча, определение промаха, сброс позиции и взаимодействие со стеной/мишенью.
- `Assets/Scripts/TargetCircle.cs` — логика бонусной мишени: очки, вспышка, случайный респаун.
- `Assets/Scripts/GameManager.cs` — общий счет, UI, milestone на 10 очков.

## Управление

- `W A S D` / `↑ ↓ ← →` — перемещение игрока.
- `F` — топспин.
- `E` — плоский удар.

## Что нужно для запуска

- Unity (версия, совместимая с проектом).
- Сцена с настроенными ссылками в инспекторе:
  - `Player`: ссылки на мяч, `Animator`, `ShotManager`,
  - `Ball`: `spawnPoint`, `GameManager`,
  - `GameManager`: `scoreText`, `wallRenderer`, материалы,
  - `TargetCircle`: коллайдер-триггер и тег мяча `Ball`,
  - у стены должен быть тег `Wall`.
